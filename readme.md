The goal of this project was to present it as a mini project for my subject ADP. 
well this was quite fun doing this project the songs.csv file was a lot of work as i manually had to add the links to spotify. 


how it works : 
step 1 -> the camera scans your face expression 
step 2 -> The model detects the face expression and finds which mood the user is (happy,sad,angry,relaxed,neutral)
step 3 -> Then based on the mood it checks the csv file in random based on the mood 
step 4 -> For each song in csv consist of a link which directs it to the spotify where the song automatially plays 

NOTE
To enable the scaning feature you need to press the button 'R' .
