## How It Works

### Step 1
The webcam continuously scans the user's facial expressions in real time.

### Step 2
The emotion detection model analyzes the face and identifies the user's mood such as:
- Happy
- Sad
- Angry
- Relaxed
- Neutral

### Step 3
Based on the detected mood, the system searches the CSV dataset and randomly selects a song related to that emotion.

### Step 4
Each song entry in the CSV file contains a Spotify link.  
The selected song link automatically opens in Spotify and starts playing.

### Continuous Detection
The system runs continuously in real time by repeatedly capturing webcam frames and updating recommendations based on the user's current emotion. 

NOTE
To enable the scaning feature you need to press the button 'R' .
